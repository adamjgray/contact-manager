contact-manager
===============

Marionette.js demo app for [indy.js](http://www.meetup.com/indyjs/) - simple contacts manager

Dependencies
------------

 - [node.js](http://nodejs.org/)

Installing
----------

    > git clone https://github.com/trip42/contact-manager.git
    > npm install
    > node server
